LeetCode-112.路径总和

发表于 2020-02-02 分类于 算法

题目描述

给你二叉树的根节点root和一个表示目标和的整数targetSum。判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和targetSum。如果存在，返回true；否则，返回false。

叶子节点 是指没有子节点的节点。

示例2

提示

  • 树中节点的数目在范围 [0, 5000] 内
  • -1000 <= Node.val <= 1000
  • -1000 <= targetSum <= 1000

题解

  1. 递归法

    将路径之和问题拆解为左子树或右子树是否有和为目标值与当前节点值之差的路径

  2. 遍历法

    将路径之和问题拆解为到达当前节点时路径上的和与左子树或右子树的和相加等于目标值的问题
