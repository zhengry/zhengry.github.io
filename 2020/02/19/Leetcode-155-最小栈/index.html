Leetcode-155.最小栈

发表于 2020-02-19 分类于 算法

题目描述

设计一个支持 push，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

MinStack() 初始化堆栈对象。
void push(int val) 将元素val推入堆栈。
void pop() 删除堆栈顶部的元素。
int top() 获取堆栈顶部的元素。
int getMin() 获取堆栈中的最小元素。

示例：

提示：

  • -231 <= val <= 231 - 1
  • ``pop、top和getMin` 操作总是在 非空栈 上调用
  • push, pop, top, and getMin最多被调用 3 * 104 次

题解

思路：

一个栈用来存当前栈上的数据，借助另一个栈来记录每次push或pop后栈上的最小值。

可以用数组的头元素做栈顶，也可以用数组的尾元素做栈顶。

使用数组的末尾元素更节省开销。
