扩大button可点击范围

发表于 2014-09-23 | 分类于 OC

在iOS开发中，经常会遇到需要在navigationBar上添加各种按钮，在相近的位置放多个按钮时，往往需要改小按钮的frame。如果按钮小于44x44，则常常会引发按钮不方便点击的情况，这时候可以重写button的pointInside方法使按钮热区不够44×44大小的先自动缩放到44×44，来扩大按钮的点击范围，再判断触摸点是否在新的热区内。
