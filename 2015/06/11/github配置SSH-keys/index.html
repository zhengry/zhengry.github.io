github配置SSH keys

发表于 2015-06-11 | 分类于 Git

实操步骤：

  1. 判断是否已有SSH

    cd ~/.ssh

  2. 删掉之前的SSH keys

    Finder 前往文件夹~/.ssh 删掉id_rsa和id_rsa.pub文件

  3. 配置git名

    git config --global user.name "zhengruyan"

  4. 配置git邮箱

    git config --global user.email "your_email_address"

  5. 重新生成SSH keys

    ssh-keygen -t rsa -C "you_email_address"

  6. 复制SSH keys：

    cat ~/.ssh/id_rsa.pub

    将回车后得到的字符串copy：

  7. 添加SSH keys到你的gitbub

    登录你的gitbub，在account setting中添加SSH keys ，名字可以随意 粘贴你复制的SSH keys，搞定！
